AGENT TAB

A spending tab for your
coding agent.

Local-first cost + bloat receipts for Claude Code and Codex. No account, no upload, no API key — it all runs on your machine.

$ npx agent-tab install View on GitHub
Then use your agent as usual and run npx agent-tab report.

What you get

Accurate cost

Real token usage from the agent's own transcript — deduped per request, with the cache read/write split. Labeled estimated, never faked.

Bloat score

A transparent 0–100 score for how wasteful a run was, capped per dimension so nothing dominates.

Waste detectors

Re-read files, lockfile reads, failing-test loops, dependency creep, huge reads, edit churn, and more — each with a concrete fix.

Rules that stick

agent-tab fix writes the recurring wastes into CLAUDE.md / AGENTS.md so the next run is leaner.

Shareable receipts

Export a card as SVG, PNG, or HTML. The painful, funny, useful thing you post.

Weekly bill

agent-tab summary totals your recent runs locally. Plus per-run history.

How it works

Claude Code / Codex
   │  lifecycle hooks (SessionStart, PreToolUse, PostToolUse, Stop, …)
   ▼
agent-tab hook  ──►  .agent-tab/runs/<session>.jsonl
   ▼
analyzer  ──►  transcript/rollout tokens + git diff + detectors
   ├─►  receipt      agent-tab report
   ├─►  weekly bill  agent-tab summary
   ├─►  share card   agent-tab share --png
   └─►  agent rules  agent-tab fix

Codex too

$ npx agent-tab install --codex

Same flow. Tokens come from Codex's on-disk rollout. Note: Codex skips a new command hook until you trust it — run /hooks inside Codex once after installing.
